## How it works

<Steps>
  <Step title="Setup review">
    Create a review template with our no-code editor for the type of content to be reviewed and the input to be captured.
  </Step>
  <Step title="Send request">
    Request a human review using any of our integrations, referencing the template and including all relevant content.
  </Step>
  <Step title="Human review">
    Pending items appear in our Agent Inbox or can be opened directly via one of the notification options.
  </Step>
  <Step title="Handle webhook">
    Completed reviews are sent to your workflow via webhook so it can continue with the provided feedback.
  </Step>
</Steps>
